이벤트

  1. 이벤트의 종류

    - 마우스 이벤트 / 키보드 이벤트 / Element 이벤트 / 사용자 이벤트

  2. 이벤트 단계

    2-1. Capturing

    2-2. Targeting

    2-3. Bubbling

  3. 이벤트 함수

    - on() 함수

    - $(대상).on('이벤트 종류', 함수);

  4. 이벤트 이름

    - blur : 포커스가 잃었을 때(마우스가 다른 곳을 향했을 때)

    - click : 마우스를 클릭했을 때

    - focusin / focusout : 포커스를 얻었을 때 / 잃었을 때
      ** 키보드의 TAB키를 사용해서 순차적으로 포커스를 이동시킬 수 있음.
      ** 포커스가 이동될 때, 기존에 포커스가 있던 요소는 포커스를 잃게 되고, 다음 요소는 포커스를 얻게 되는 이벤트가 동시에 발생.

    - change : 셀렉트 박스의 목록을 선택 했을 때

    - keydown / keyup : 키보드를 눌렀을 때 / 떼었을 때

    - mousedown / mouseup : 마우스 버때튼 눌렀을 때 / 떼었을 때

    - mouseenter / mouseleave : 마우스 커서가 닿았을 때 / 떨어졌을 때 (bubbling 발생하지 않음)

    - mouseover / mouseout : 마우스 커서가 닿았을 때 / 떨어졌을 때 (bubbling 발생함 => 사용하지 않는 것이 좋음)

    - resize : 브라우저의 크기가 변경 될 때(BOM)

    - scroll : 스크롤 될 때 (보통은 웹페이지 전체 스크롤 - window 객체)

    - load / unload : 대상이 로딩 되었을 때 / 제거 되었을 때 (이미지가 모두 로딩된 이후에 코드가 실행되게 할 때 주로 사용)

    - select : 텍스트를 마우스로 드래그해서 선택 했을 때
